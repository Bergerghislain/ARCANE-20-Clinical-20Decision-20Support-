# Shortlist d'offres junior / new grad a cibler

Recherche effectuee le 9 juin 2026. A reverifier sur les pages carrieres avant candidature, car les offres ferment vite.

## Priorite 1 - Tres bon fit et bon nom

### 1. Mistral AI - Software Engineer, New Grad

- **Type** : full-time, new grad / 0-2 ans.
- **Lieu** : Paris, onsite/hybrid.
- **Fit estime** : 95%.
- **Pourquoi** : Python/TypeScript, AI platform, LLM, infra, tests, produit. C'est la cible la plus naturelle.
- **Message CV** : early-career engineer, Python/TS, LLM/ARGOS, Docker/CI, product mindset.
- **Action** : priorite absolue, utiliser le reseau alumni Ensimag.

### 2. Dataiku - Fullstack Software Engineer, Core

- **Type** : full-time, niveaux junior/mid/senior.
- **Lieu** : Paris/hybrid ou remote France selon offre.
- **Fit estime** : 88-92%.
- **Pourquoi** : vrai produit data/AI, stack agnostic, Python, Java, Angular, MLOps, data preparation.
- **Message CV** : ARCANE comme produit data/IA complet : backend, frontend, data, LLM, tests.
- **CV associe** : `cv_dataiku_fullstack_core.tex`.

### 3. Algolia - Junior Software Engineer, AI Platform / AI Search

- **Type** : junior, souvent 1+ an experience.
- **Lieu** : Paris / remote France selon offre.
- **Fit estime** : 82-88%.
- **Pourquoi** : AI platform/search, Python ou Go, APIs, data pipelines, cloud, testing.
- **Forces de ton profil** : Python/FastAPI, API services, LLM, PostgreSQL, tests, Azure fundamentals.
- **Gaps** : Go, data pipelines a grande echelle, Kubernetes selon equipe.
- **Action** : tres bonne cible. Adapter le CV Dataiku vers "AI Platform / data pipelines".

### 4. Doctolib - Pipeline Software Engineer Fullstack Java / React

- **Type** : full-time, plusieurs seniorities dont early-career possible.
- **Lieu** : Levallois-Perret / Paris, hybrid.
- **Fit estime** : 78-82%.
- **Pourquoi** : sante, React/TypeScript, produit, tests, securite.
- **Gaps** : JVM/Spring Boot.
- **CV associe** : `cv_doctolib_pipeline_fullstack.tex`.

### 5. Padoa - Developpeur Full Stack Junior, Squad IA

- **Type** : CDI junior, 0-2 ans.
- **Lieu** : Paris, remote frequent.
- **Fit estime** : 80-85%.
- **Pourquoi** : Python, Node/TypeScript, PostgreSQL, Angular, GenAI, monitoring qualite/cout/latence.
- **Gaps** : Angular a renforcer, monitoring IA a expliciter.
- **Action** : deja une excellente cible.

## Priorite 2 - Bon nom, fit correct, a tenter selon energie

### 6. Amazon - 2026 Graduate Software Development Engineer

- **Type** : graduate.
- **Lieu** : selon pays/equipe, pas forcement France.
- **Fit estime** : 78-85%.
- **Pourquoi** : accepte jeunes diplomes, langages generalistes Java/Python/C#/TypeScript, CS fundamentals.
- **Forces** : Ensimag, algorithms/OOP, Python/Java/C#, projets full-stack.
- **Gaps** : preparation intensive OA + system design leger + leadership principles.
- **Action** : interessant si tu es ouvert a l'international ou si une localisation France/Europe pertinente apparait.

### 7. Google - Software Engineer II, Machine Learning / early-career Paris

- **Type** : pas new grad pur ; demande souvent 1+ an experience + ML.
- **Lieu** : Paris selon posting.
- **Fit estime** : 65-75%.
- **Pourquoi** : tres bon nom, profil Python/AI possible.
- **Gaps** : barre algo/system design elevee, ML concepts a demontrer, processus tres selectif.
- **Action** : candidature possible mais preparation LeetCode/ML obligatoire.

### 8. Criteo - Software Engineer Intern / Campus / Software Development Engineer

- **Type** : stages 6 mois et postes permanents souvent 2-3 ans.
- **Lieu** : Paris / Grenoble.
- **Fit estime** : 70-80% selon offre.
- **Pourquoi** : tres bon nom engineering en France, web apps, C#/TypeScript/Angular, scale, data.
- **Gaps** : certaines offres sont stages fin d'etudes uniquement ; SDE permanent demande parfois 2-3 ans.
- **Action** : a surveiller. Fort si tu peux encore faire un stage, sinon attendre une offre graduate/junior.

### 9. Alan - Software Engineer Internship / early product engineering

- **Type** : stage fin d'etudes dans les resultats vus ; full-time a surveiller.
- **Lieu** : Paris.
- **Fit estime** : 75-85% si eligibilite stage, sinon a surveiller.
- **Pourquoi** : Python/Flask, React, PostgreSQL, produit sante/assurance, ownership.
- **Gaps** : stage demande souvent d'etre encore inscrit en derniere annee.
- **Action** : verifier eligibilite. Si pas eligible, chercher une ouverture full-time.

### 10. Bigblue - Graduate Product Engineer

- **Type** : graduate/product engineering.
- **Lieu** : Paris.
- **Fit estime** : 70-75%.
- **Pourquoi** : top ecole, SQL, support produit, operations, possibilite dev.
- **Gaps** : moins software-engineering pur que Mistral/Dataiku/Doctolib.
- **Action** : bonne option si tu veux un role plus produit/ops avec du code.

## Priorite 3 - Bon nom mais pas la meilleure efficacite de candidature

### 11. Back Market - Software Engineer Payment / Associate Backend

- **Type** : backend, souvent 2+ ans pour associate ou plus pour senior.
- **Lieu** : Barcelone, hybrid.
- **Fit estime** : 68-72%.
- **Pourquoi** : Python/FastAPI, PostgreSQL, APIs, reliability, payment systems.
- **Gaps** : Go, Celery/Kafka/RabbitMQ, payment domain, distributed systems, Barcelone.
- **CV associe** : `cv_backmarket_payment.tex`.
- **Action** : postuler seulement si tu es motive par payment/marketplace et pret a preparer system design.

### 12. Hugging Face - internships / open-source ML roles

- **Type** : roles tres variables ; souvent senior ou ML/research specific.
- **Lieu** : remote / Paris.
- **Fit estime** : 60-80% selon role.
- **Pourquoi** : LLM/open-source/AI, excellent nom.
- **Gaps** : beaucoup d'offres demandent forte experience ML/open-source.
- **Action** : a surveiller. Postuler si un role "internship", "new grad", "developer tools", "AI applications" apparait.

### 13. Contentsquare - Fullstack / AI Fullstack

- **Type** : souvent mid/senior dans les resultats vus.
- **Lieu** : Paris / remote Europe.
- **Fit estime** : 60-70% actuellement.
- **Pourquoi** : TypeScript/React/Node/Python, produit data/analytics.
- **Gaps** : 4-5 ans demandes sur plusieurs offres.
- **Action** : surveiller junior/internship. Pas prioritaire si l'offre demande explicitement 4+ ans.

### 14. Yield Studio & Advisory - Full Stack AI React / Python

- **Type** : CDI, 2-5 ans dans les resultats vus.
- **Lieu** : Paris.
- **Fit estime** : 72-78%.
- **Pourquoi** : React/Python/LLM/RAG/agents/PostgreSQL.
- **Gaps** : cabinet/studio plus petit, demande souvent 2+ ans.
- **Action** : bon plan B si tu veux multiplier les candidatures IA full-stack.

## Strategie de candidature

### Ne pas envoyer le meme CV partout

- **Mistral / Padoa / Algolia / Dataiku** : version AI/data/platform.
- **Doctolib / Alan** : version healthcare/product/full-stack.
- **Back Market / Amazon** : version backend/API/reliability/CS fundamentals.
- **Criteo** : version C#/TypeScript/data/scale.

### Ce qui doit apparaitre partout

- Ensimag.
- Projet ARCANE presente comme produit reel.
- React/TypeScript + Python/FastAPI + PostgreSQL.
- Tests + CI/CD.
- Securite / donnees sensibles.
- Capacite a apprendre une stack existante sans ego.

### Ce qu'il ne faut pas faire

- Ne pas dire "expert Spring Boot" si ce n'est pas vrai.
- Ne pas dire "experience Kafka/Kubernetes production" si ce n'est pas vrai.
- Ne pas mettre l'IA au centre pour Back Market.
- Ne pas postuler a trop d'offres seniors : il vaut mieux 8 candidatures tres ciblees que 50 candidatures faibles.
